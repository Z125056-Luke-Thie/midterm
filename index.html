Production Philosophy

Function Before Form

I begin all my web projects from my own made blueprint. This blueprint contains the standard index.php file, bootstrap, images folder, as well as a javascript folder, and a rows folder.

I make all my pages out of rows, each piece of content is a new "row" which is then with PhP required on the correct webpage. This makes having just 1 header and footer much easier, and you can recycle content where necessary.

Besides the blueprint, I try to make all my web-projects to be handbuild, and not to be constrained by any frameworks.
